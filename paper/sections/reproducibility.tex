\section{Reproducibility}
\label{sec:reproducibility}

We are committed to open and reproducible research. This section documents resources for reproducing our analysis.

\subsection{Data Availability}
\label{sec:reproducibility:data}

\paragraph{Moltbook Observatory Archive.} Our primary dataset is publicly available on Hugging Face:
\begin{center}
\url{https://huggingface.co/datasets/SimulaMet/moltbook-observatory-archive}
\end{center}
The dataset is released under the MIT license. We provide a download script (\texttt{scripts/download\_moltbook\_observatory\_archive.py}) that fetches and exports the required tables to local storage.

\paragraph{Reddit comparison data.} Due to Reddit's terms of service, we cannot redistribute raw Reddit data. Instead, we provide:
\begin{itemize}
    \item Post and comment IDs enabling re-collection via Reddit's API,
    \item Pre-computed features (timestamps, parent links, anonymized author hashes) sufficient for reproducing our analyses.
\end{itemize}

\paragraph{Processed datasets.} We release processed intermediate datasets (thread trees, survival times, computed metrics) in Parquet format to enable reproduction without re-running preprocessing.

\subsection{Reproducing Key Results}
\label{sec:reproducibility:reproduce}

To reproduce our main findings:

\begin{enumerate}
    \item \textbf{Environment setup:}
    \begin{verbatim}
    python -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    \end{verbatim}

    \item \textbf{Data download:}
    \begin{verbatim}
    python scripts/download_moltbook_observatory_archive.py \
        --out-dir data/raw/moltbook-observatory-archive
    \end{verbatim}

    \item \textbf{Preprocessing:}
    \begin{verbatim}
    python analysis/01_preprocess.py
    \end{verbatim}

    \item \textbf{Main analyses:}
    \begin{verbatim}
    python analysis/02_geometry.py      # Conversation geometry
    python analysis/03_halflife.py      # Half-life estimation
    python analysis/04_periodicity.py   # Spectral analysis
    python analysis/05_comparison.py    # Cross-platform comparison
    \end{verbatim}

    \item \textbf{Generate paper figures:}
    \begin{verbatim}
    python analysis/06_figures.py
    \end{verbatim}
\end{enumerate}

Expected outputs are documented in \texttt{outputs/README.md}.
